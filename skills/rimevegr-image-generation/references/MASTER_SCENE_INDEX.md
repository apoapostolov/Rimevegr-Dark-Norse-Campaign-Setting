# Master scene index

This is the high-coverage scene taxonomy for the Rimevegr image-generation
skill. Use it as the master menu for building prompts and prompt batches.

Goal:

- avoid a chaotic sprawl of one-off prompt files
- ensure broad coverage so almost no practical scene is left unclassified
- make it easy to assemble JSON or YAML batches from modular scene logic

## How to use this index

Pick one item from each major layer:

1. scene family
2. sub-scene or action type
3. social or material pressure
4. visual mode
5. optional supernatural pressure only if explicitly requested
6. color family and emotion

Do not fill every slot at once. Use this index as a routing system, not as a
prompt to dump whole.

## Suggested master routing map

Use this logic when deciding whether a dedicated ready-made batch is worth
keeping:

### Keep as reusable ready batch

- major recurring character packs
- settlement identity packs
- pantheon symbol packs
- landmark sets
- high-value mood panoramas
- continuity-critical faction scenes

### Build on demand from the index

- one-off tavern scenes
- one-time bargaining scenes
- rare weather combinations
- specific travel moments
- niche aftermath shots
- temporary concept variations

## Coverage note

This file is meant to be hyper-extensive enough to route almost any indoor,
outdoor, battle, tavern, ritual, labor, travel, portrait, panorama, or dread
scene the user may request later.

If a future request does not fit the index cleanly, add the missing class here
before creating many one-off prompt files.
